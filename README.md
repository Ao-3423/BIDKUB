# BIDKUB

**BIDKUB** คือ Personal Pocket Book สำหรับจัดการค่าใช้จ่ายร่วมกัน สามารถเพิ่มสมาชิก, เพิ่มรายการค่าใช้จ่าย, คำนวณยอดคงเหลือ และสร้างรายการชำระหนี้อัตโนมัติ

## Features
- เพิ่ม/จัดการสมาชิก
- เพิ่มรายการค่าใช้จ่าย พร้อมวิธีหาร:
  - หารเท่า
  - เลือกคนที่หาร
  - กำหนดสัดส่วนเอง
- แสดงรายการทั้งหมด
- สรุปยอดคงเหลือของแต่ละคน
- สร้างรายการชำระหนี้อัตโนมัติ
- นำเข้า/ส่งออก JSON
- ล้างข้อมูลทั้งหมด

## Tech Stack
- HTML / CSS / JavaScript
- ใช้ [SweetAlert2](https://sweetalert2.github.io/) สำหรับ Popup

## Usage
1. เปิด `index.html` บนเว็บเบราว์เซอร์
2. เพิ่มสมาชิก และเริ่มเพิ่มรายการค่าใช้จ่าย
3. ตรวจสอบยอดคงเหลือ และดูรายการชำระหนี้
